== Description ==
Whitesite theme toolkit is a basic set of editors for Whitesite themes.


== Installation ==
Unzip data to wp-content/plugins/whitesite-theme-toolkit
Enable in plugins->instaled plugins

* Basic object structurest: button, current-article, image, jumbotron (unfinished), portfolio, slider, swipe-card, text, text-image-link, video-background
* Front page editor
* Page/Post internal editor
